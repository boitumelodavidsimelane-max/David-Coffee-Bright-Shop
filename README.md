Bright Coffee Shop Sales Analysis

# 📖 Project Overview
This repository contains an analysis of the **Bright Coffee Shop Sales** dataset, which captures daily transactional information.  
The business has recently appointed a new CEO whose mission is to **grow revenue** and **improve product performance**.  
As a Junior Data Analyst, my role is to extract actionable insights from historical data and prepare a presentation to assist the CEO in decision-making.

---

#🎯 Objectives
- Identify revenue trends and growth opportunities.
- Evaluate product performance across categories.
- Highlight customer purchasing behavior.
- Provide clear, actionable recommendations for the CEO.

---

# 📊 Key Insights
- **Revenue Trends:** Seasonal peaks and weekday vs weekend differences.
- **Top Products:** Certain beverages consistently outperform others.
- **Customer Behavior:** Bundling patterns suggest upsell opportunities.
- **Growth Opportunities:** Introducing loyalty programs and optimizing underperforming products.

---

# 📂 Repository Contents
- `data/` → Raw dataset.

- `src/` → Python scripts for cleaning, visualization, and analysis.
- `reports/` → Final presentation and dashboards for the CEO.
